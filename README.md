# sunburn.io

## Stack

* Ruby 2.1.2
* Middleman 3.x


## Getting started

Make sure you're running Ruby 2.1.2, and install the bundle with:

```sh
$ bundle install
```

Once you've done that, you can start the server with:

```sh
$ bundle exec middleman server
```

The site will be served up on `http://localhost:4567`.
